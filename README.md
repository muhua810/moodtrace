## 💡 项目背景

当代大学生面临学业、就业、人际关系等多重压力，心理健康问题日益受到关注。然而大多数心理自助工具存在以下痛点：

| 痛点 | 传统方案 | 情绪日历的解决方案 |
|------|---------|-----------------|
| 门槛太高 | 需要长篇书写情绪日记 | **一句话记录**，3 秒完成 |
| 缺乏洞察 | 只记录不分析 | **AI 自动分析** + **可视化洞察** |
| 隐私顾虑 | 数据上传云端 | **纯前端存储**，AES-256-GCM 加密 |
| 缺少关怀 | 无后续跟进 | **3 级渐进式关怀** + 心理热线 |

## ✨ 核心功能

- ✍️ **每日情绪记录** — 一句话描述今日感受，支持中文自然语言
- 🤖 **AI 情绪分析** — 四级降级策略：Workers AI → 用户 API → 本地关键词引擎 → 统计分析器
- 📊 **情绪热力图日历** — 自研 SVG 组件，GitHub 贡献图风格，年/月双视图
- 📈 **多维度统计** — 饼图/趋势图/柱状图 + 工作日/周末分析 + 年度报告
- 💛 **渐进式关怀** — 3 级关怀系统：鼓励 → 放松建议 + 热线 → 紧急提示
- 🔒 **隐私优先** — 纯前端 + AES-256-GCM 加密存储，数据不出设备
- 📱 **PWA 支持** — 离线缓存、可安装到手机桌面
- ☁️ **云端备份** — 基于设备 ID 的无注册备份/恢复
- ♿ **无障碍** — 44px 触摸热区、键盘操作、屏幕阅读器兼容

## 🔬 技术创新

### 创新点 1：中文情绪四级分析引擎

```
┌─────────────────────────────────────────────────────┐
│               情绪分析四级降级策略                      │
├─────────────┬───────────────────────────────────────┤
│ Level 1     │ Workers AI 代理（DeepSeek）             │
│ Level 2     │ 用户自定义 OpenAI 兼容 API              │
│ Level 3     │ 本地关键词引擎（140+ 关键词）            │
│ Level 4     │ TF-IDF + 朴素贝叶斯统计分析器            │
└─────────────┴───────────────────────────────────────┘
```

**本地关键词引擎核心特性**：

- **分句加权算法**：检测"虽然…但是…"反转模式，后半句权重 2.5x
- **否定词检测**：29 个否定词，10 字符检测窗口，长词优先匹配
- **反讽检测**：6 种模式（"呢"结尾 + 正面词、"呵呵"、"真是太X了呢"等）
- **网络用语**：yyds、emo、破防、蚌埠住了、摆烂、躺平等 20+ 条
- **Emoji 映射**：80+ emoji 情绪映射

```javascript
// 示例：「虽然今天很累，但是完成了任务很开心」
// → 检测到反转模式
// → "很累"权重 0.5x，"很开心"权重 2.5x
// → 结论：positive（心情不错）
```

### 创新点 2：情绪热力图日历

自研 SVG 组件，复用开发者社区熟悉的 GitHub 贡献图视觉语言：
- 12 个月标签 + 7 行星期标签
- 年/月双视图切换
- 选中发光效果 + Tooltip 浮层
- 响应式设计（移动端自动缩小格子）
- 44px 触摸热区（WCAG 标准）

### 创新点 3：3 级渐进式关怀系统

```
Level 1（1-2天低落）→ 温柔鼓励 + 放松建议
Level 2（3-4天低落）→ 深度关怀 + 心理热线
Level 3（5+天低落）→ 紧急提示 + 心理热线高亮
```

同时检测危机关键词（如"想死"）直接触发最高级关怀。**用代码传递温度**。

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/muhua810/mood-calendar.git
cd mood-calendar

# 安装依赖
npm install

# 启动开发服务器
npm run dev        # → http://localhost:3000

# 运行测试
npm test                    # 一次性运行
npm run test:coverage       # 覆盖率报告

# 构建
npm run build
```

### 部署到 Cloudflare

```bash
# 前端（Cloudflare Pages）
npx wrangler pages deploy dist --project-name=mood-calendar

# 后端（Cloudflare Workers）
cd worker
npx wrangler kv namespace create MOOD_STATS
npx wrangler deploy
```
