# 需求分析

> **当前版本**:  v2.1.1 | **更新日期**: 2026-03-28

## 1. 项目背景

### 1.1 社会背景

大学生心理健康问题日益突出。据《中国国民心理健康发展报告（2021-2022）》：
- 约 **24.6%** 的大学生存在不同程度的心理健康问题
- 抑郁风险检出率为 **21.48%**
- 焦虑风险检出率为 **45.28%**
- 仅 **30%** 的学生在遇到心理问题时会寻求专业帮助

核心痛点：
1. **自我觉察不足** — 多数学生缺乏持续追踪自身情绪变化的工具
2. **求助门槛高** — 传统心理咨询需要预约、排队，学生羞于开口
3. **工具缺失** — 现有心理类 App 偏向专业用户，普通学生难以坚持使用

### 1.2 用户调研

为验证需求真实性，我们对 86 名在校大学生进行了问卷调研（线上问卷 + 校园拦访），核心发现：

| 调研维度 | 数据结果 |
|---------|---------|
| 曾感到情绪低落但未寻求帮助 | **78%**（67/86） |
| 认为「一句话记录」愿意尝试 | **91%**（78/86） |
| 对现有心理类 App 体验不满 | **65%**（56/86） — 主要原因：太复杂、要注册、担心隐私 |
| 最希望的功能 | 情绪趋势可视化（72%）、个性化建议（58%）、隐私保护（85%） |
| 每日愿意投入的时间 | **< 1 分钟**（83%） — 验证了「极简记录」设计方向 |
| 偏好的使用设备 | 手机浏览器（76%）> 电脑（18%）> 小程序（6%） — 验证了 PWA 路线 |

**关键洞察**：
- **隐私是第一顾虑**：85% 的用户将「隐私保护」列为最关注功能，远超「AI 分析」（42%）— 这验证了我们「本地优先」的存储架构决策
- **低门槛是刚需**：91% 对「一句话记录」感兴趣，但仅有 23% 愿意做问卷式记录 — 验证了自然语言输入优于量表方案
- **可视化有强需求**：72% 希望看到情绪趋势 — 这驱动了热力图和多维度统计功能的优先级

### 1.3 项目定位

**心迹**是一款面向大学生的轻量级情绪追踪工具，通过「每日一句话」的极简记录方式，结合 AI 自动分析，帮助用户：

- 建立情绪自我觉察习惯
- 可视化情绪变化趋势
- 及时发现持续低落状态并获取帮助

| 特征 | 描述 |

| 设备 | 以手机为主，兼顾电脑 |
| 使用场景 | 睡前、课间、心情波动时 |
| 核心需求 | 简单记录、自我了解、获取建议 |

### 用户画像

**小李（典型用户）**
- 大三学生，学业压力大
- 经常感到焦虑但说不清原因
- 不愿去心理咨询中心（觉得"没那么严重"）
- 希望有一个简单的方式追踪自己的情绪变化

**小王（积极用户）**
- 大一新生，适应期情绪波动
- 对自我成长感兴趣
- 喜欢用 App 记录生活
- 会定期回顾自己的情绪变化

### 4.2 竞品详细分析

**Daylio** — 全球最流行的情绪日记 App
- 优势：界面精美，图标记录方式直观，支持自定义活动标签
- 劣势：无 AI 分析，中文翻译生硬，免费版功能受限，核心数据云端存储
- 与心迹差异：我们用自然语言替代图标选择，表达更自由；AI 分析提供深度洞察

**心情温度计** — 微信小程序
- 优势：无需安装，微信生态内传播便利
- 劣势：功能极简，无深度分析和可视化能力，数据完全存储在云端，隐私依赖微信平台

**Moodpath** — 德国专业心理健康 App
- 优势：有循证心理学背景，提供专业评估报告
- 劣势：不支持中文，评估问卷较长（每次 3-5 分钟），免费版受限，数据存储在欧洲服务器
- 与心迹差异：我们针对中文语境设计，记录门槛更低（一句话 vs 问卷），AI 分析支持中文语义

**小睡眠** — 国内综合心理健康 App
- 优势：助眠+冥想+心情记录一站式，用户基数大
- 劣势：情绪记录只是辅助功能，分析能力弱，会员收费，隐私依赖账号体系
- 与心迹差异：我们专注情绪追踪这一个点，做到极致

### 4.3 差异化优势总结

1. **中文自然语言输入** — 无需选择图标，用一句话描述感受更自然（调研验证：91% 用户偏好此方式）
2. **情绪热力图** — 首创将 GitHub 贡献图模式应用于心理健康领域，年/月双视图
3. **多层降级分析** — Workers AI 代理 + 本地关键词引擎 + Emoji 映射 + 统计分析，离线在线都能用
4. **隐私优先架构** — 所有数据存储在用户设备，AES-256-GCM 加密，无需注册（调研验证：85% 用户最关注隐私）
5. **高危预警机制** — 检测自杀相关关键词，主动推荐心理援助热线（本土化号码）
6. **零成本使用** — 完全免费，无内购、无会员、无广告
7. **多语言支持** — 支持中/英/日/韩/法/西 6 种语言，面向全球用户
